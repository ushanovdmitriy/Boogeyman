# Boogeyman

Telegram-бот, который собирает историю чата за последний месяц, анализирует её локально и с помощью Claude (Anthropic API), и генерирует красивый HTML-отчёт.

## Что делает бот

- Собирает сообщения из указанного чата за последние 30 дней через Telethon (User API)
- Считает локальную статистику: топ участников, активность по дням, популярные слова
- Отправляет сводку в Claude для AI-анализа: темы, тональность, ключевые события
- Рендерит всё в HTML-отчёт и присылает файл в Telegram

## Получение ключей

**TELEGRAM_BOT_TOKEN** — создать бота у [@BotFather](https://t.me/BotFather), команда `/newbot`

**TELEGRAM_API_ID / TELEGRAM_API_HASH** — зайти на [my.telegram.org](https://my.telegram.org), раздел "API development tools", создать приложение

**ANTHROPIC_API_KEY** — получить на [console.anthropic.com](https://console.anthropic.com)

**CHAT_ID** — id целевого чата или супергруппы (для супергрупп начинается с `-100`)

## Установка и запуск

```bash
pip install -r requirements.txt
cp .env.example .env
# заполнить .env своими ключами
python bot.py
```

### Первый запуск Telethon

При первом запуске Telethon запросит номер телефона и код авторизации из Telegram — это нужно для доступа к истории чата от имени пользователя (не бота). После успешной авторизации создаётся файл сессии `*.session`, который сохраняется локально и повторная авторизация не требуется.

## Использование

После запуска отправьте боту:

- `/start` — инструкция
- `/report` — запустить генерацию отчёта
